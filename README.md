# projeto-calculadora

1. Soma
2. Subtração
3. Multiplicação
4. Divisão

2. Fonte Poppins

1. Criação do Arquivo HTML
2. Criação do Arquivo CSS
3. Criação dos Botões da Calculadora
4. Criação do Arquivo JavaScript
5. Criação das Funções em JavaScript
6. Criação dos Eventos nas tags de HTML
7. Utilização do Document que serve para acessar o HTML(DOM)
